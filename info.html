palette What is OKLCH?

OKLCH is a modern color model that represents colors exactly as the human eye perceives them. Unlike RGB, it is perceptually uniform.

  • L (Lightness): Perceived lightness from black to white.
  • C (Chroma): The intensity and purity of the color.
  • H (Hue): Position on the color wheel (0-360°).

help How to use

1. Click Generate (or press SPACE) to create a new palette.

2. Use the lock icon to save the colors you like.

3. Adjust L, C, H parameters for precise tuning.
